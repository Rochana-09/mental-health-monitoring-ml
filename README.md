# Mental Health Monitoring & Wellness Recommendation System

## Project Overview
This project presents an AI-based Mental Health Monitoring and Wellness Recommendation System designed to analyze behavioral and physiological data to detect stress patterns and provide personalized wellness suggestions. 

The system leverages machine learning algorithms to identify mental health trends and support early detection of stress-related conditions.

---

## Objectives
- Monitor mental health indicators using user behavioral data
- Apply machine learning algorithms to detect stress patterns
- Provide personalized wellness recommendations
- Visualize trends and insights for better understanding

---

## Dataset
The dataset contains user behavioral and lifestyle attributes such as:

- Sleep hours
- Work hours
- Physical activity
- Mood level
- Stress level

The dataset is used to train machine learning models for stress prediction.

---

## Project Workflow

1. Data Collection  
2. Data Cleaning and Preprocessing  
3. Feature Engineering  
4. Exploratory Data Analysis  
5. Model Training  
6. Model Evaluation  
7. Prediction and Recommendation

---

## Machine Learning Models Used

- Support Vector Machine (SVM)
- Logistic Regression
- LightGBM
- XGBoost
- CatBoost

A stacking ensemble model was implemented to improve prediction accuracy.

---

## Model Evaluation Metrics

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

---

## Results

The ensemble model achieved high accuracy in predicting stress levels based on behavioral data.

The results demonstrate the effectiveness of machine learning in identifying mental health patterns.
